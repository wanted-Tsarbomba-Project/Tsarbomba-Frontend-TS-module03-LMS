# v0.2.0 (2026-06-29)

### 관리자 뱃지 관리

- **Added**
  - 관리자 뱃지 목록 조회 페이지 추가
  - 관리자 뱃지 등록 페이지 추가
  - 관리자 뱃지 수정 및 삭제 페이지 추가
  - 뱃지 이미지 업로드 및 미리보기 기능 추가
  - 뱃지 상태(`ACTIVE`, `INACTIVE`) 수정 기능 추가
  - 뱃지 등록·수정·삭제 확인 모달 추가
  - `/api/v1/admin/badges` 조회 및 등록 API 연동
  - `/api/v1/admin/badges/{badgeId}` 상세 조회, 수정, 삭제 API 연동

- **Changed**
  - 뱃지 목록을 카드형 UI로 구성하여 이미지, 이름, 설명, 필요 포인트, 상태, 등록일을 표시
  - 뱃지 등록·수정 화면을 강좌 등록 페이지와 유사한 폼 UI로 구성
  - 뱃지 목록 카드 클릭 시 수정 페이지로 이동하도록 변경
  - `/admin/badges/new`, `/admin/badges/[id]/edit` 진입 시 사이드바 뱃지 관리 메뉴 활성화

- **Fixed**
  - 뱃지 상세 조회 실패 또는 정보 없음 상태에서 안내 모달이 표시되지 않을 수 있는 문제 수정
  - 뱃지 수정 화면 모달 상태에서 성공 여부와 목록 이동 여부가 섞이던 상태값 분리
